# 🎯 InterviewEnglish AI

An AI-powered English learning and interview-prep tool built for Tamil Nadu engineering students, helping them practice spoken and written English for job interviews.

**🔗 Live demo:** https://interview-english-ai-yb3sdvslhs5tic2kzf4x28.streamlit.app/

## Features
- **Self-introduction practice** — structured prompts to build a confident intro
- **Mock interview chat** — AI-powered interview simulation with realistic questions
- **Grammar checks** — instant feedback on written English
- **Daily conversation practice** — everyday English practice scenarios
- **Progress tracking** — see improvement over time

## Tech stack
- Python
- Streamlit (UI + deployment)
- Google Gemini API (AI responses)
- Prompt Engineering

## Run it locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Why I built this
Many engineering students in Tamil Nadu are strong technically but lose interview opportunities due to lack of spoken English confidence. This tool gives them a free, judgment-free space to practice daily — self-intros, mock interviews, and grammar — before facing a real recruiter.
